Linear Transformation and Change of Basis

Linear transformations and change of basis are widely used in statistics, for this reason I briefly describe the definition of these concepts and how they are related.

Linear Transformation

Letting \(V\) and \(W\) be vector spaces, a function \(f: V \rightarrow W\) is a linear transformation if the additivity and scalar multiplication properties are hold for any two vectors \({\mathbf{u}}, {\mathbf{v}} \in V\) and a constant \(c\): \[f({\mathbf{u}}+{\mathbf{v}}) = f({\mathbf{u}}) + f({\mathbf{v}})\] \[f(c{\mathbf{v}}) = cf({\mathbf{v}}).\]

This concept is more common to use when working with matrices, considering the vector spaces \(V \in {\mathbb{R}}^n\) and \(W \in {\mathbb{R}}^m\), and a matrix \({\mathbf{A}}_{m \times n}\) leads to the linear transformation \(f({\mathbf{x}}) = {\mathbf{A}}{\mathbf{x}}\), from \({\mathbb{R}}^n\) to \({\mathbb{R}}^m\). Note that the linear transformation is relative to the basis of the vector spaces. Here, we are using the standard basis.

Eigenvectors

The eigenvector of a linear transformation \(A\) is a non-zero vector \(v\) such us the linear transformation of it is proportional to itself: \(Av = \lambda v\)

Geometrically, you can think of \(v_1, v_2, \dots, v_n\) as the basis of vectorial space of the transformation \(A\) and of \(\lambda_1, \lambda_2, \dots, \lambda_n\) as # the length in direction of the corresponding eigenvector.

There are certain properties the are useful for statistical modelling such as: - Trace of \(A\) is equals to the sum of the eigenvalues. - Determinant of \(A\) is equals to the sum of the eigenvalues. - If \(A\) is symmetric, then all eigenvalues are real. - If \(A\) is positive definite, then all eigenvalues are positive.

Some of these properties can be explained using the eigendecomposition: \(A = QVQ^{-1}\)
